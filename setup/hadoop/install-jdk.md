# Install JDK

## Update Packages
```sh
sudo apt-get update
sudo apt-get -y upgrade
```

## Install JDK (Choose Either One)

### OpenJDK
```sh
sudo apt-get -y install openjdk-8-jdk-headless
```

### Oracle JDK
```sh
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get -y install oracle-java8-installer
```
