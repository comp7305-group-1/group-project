# Xen 4.9 Setup Steps

1. [Install Xen](xen-install.md)
   - Once on Bare Metal
   - Turn Bare Metal to Dom0
   - Login as admin@bare-metal
1. [Setup Dom0](dom0-setup.md)
   - Once on Dom0
   - Login as admin@dom0
1. [Setup a New VM](domu-setup.md)
   - Every time a new DomU is created
   - Login as admin@dom0
   - Then login as root@domu
